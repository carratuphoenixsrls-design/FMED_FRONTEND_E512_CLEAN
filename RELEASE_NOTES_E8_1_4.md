# FMED Enterprise 1.0 · E8.1.4

Micro-release frontend che rende sempre visibile la gestione delle scadenze obsolete direttamente da `App_nuovo.jsx`.

- Barra amministrativa visibile in alto nella pagina Scadenze.
- Selezione di tutte le scadenze scadute filtrate.
- Cessazione controllata delle sole scadenze scadute selezionate.
- Nessuna cancellazione: storico e audit restano invariati.
- Nessuna modifica al backend E8.1.3 o a Supabase.
- Evita dipendenze dall'aggiornamento dei componenti annidati durante upload GitHub manuali.
