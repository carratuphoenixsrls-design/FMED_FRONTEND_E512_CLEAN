1. Clona FMED_FRONTEND_CLEAN con GitHub Desktop.

4. Indica il percorso del clone locale.
5. In GitHub Desktop: Commit to main, poi Push origin.
6. Vercel creerà automaticamente un nuovo deployment.

NON usare Redeploy su deployment vecchi.
NON copiare la cartella esterna dentro il repository.

vite.config.js, vercel.json e index.html.
