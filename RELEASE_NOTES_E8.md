# FMED Enterprise 1.0 — E8 Audit finale e release stabile

La E8 conclude la roadmap Enterprise e mantiene tutte le funzioni delle release precedenti.

## Novità

- sezione amministrativa **Audit finale**;
- controllo non distruttivo di Supabase, utenti, ruoli, Asset, SharePoint, cicli, processi e Master Data;
- esito generale `CONFORME`, `CON_RILIEVI` o `NON_CONFORME`;
- punteggio e dettaglio di ogni controllo;
- export del report in JSON;
- endpoint di readiness;
- header HTTP di sicurezza;
- finestra operativa dal 01/01/2023 invariata;
- tutti i collaudi sempre conservati.

Nessun SQL Supabase richiesto.
